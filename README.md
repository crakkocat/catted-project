# 🐱 Catted Project
A fork of ReVanced, enhanced and refined.
📌 Overview
Catted Project is a custom fork of ReVanced, focused on delivering a streamlined, customizable, and user-friendly experience for patched Android apps.
Built with flexibility in mind, Catted Project aims to provide improved usability, cleaner configuration, and additional enhancements beyond the base ReVanced ecosystem.
✨ Features
🔧 Advanced patch customization
🚀 Improved performance and stability
🎨 Cleaner and more intuitive configuration
📦 Preconfigured patch sets (optional)
🔄 Regular updates synced with upstream ReVanced
🐱 Catted-specific enhancements and tweaks
📥 Installation
Prerequisites
Android device (Android 8.0 or higher recommended)
Basic knowledge of APK installation
Unknown sources enabled
Steps
Download the latest release from:
👉 https://github.com/crakkocat/catted-project/releases⁠�
Install the required components:
Catted Manager (if applicable)
MicroG (for apps requiring Google services)
Select your target app (e.g., YouTube)
Apply patches using the Catted interface
Install the patched APK
🛠️ Usage
Open Catted Manager
Choose an app to patch
Select desired patches
Build the patched APK
Install and enjoy 🎉
🔄 Updating
To update:
Download the latest version of Catted Project
Re-patch your apps with updated patches
Reinstall patched APKs
